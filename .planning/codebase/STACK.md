# Stack

## Core Technologies
- **Python 3.10+**: Main brain for AI agents and worker logic.
- **Next.js 15 (React 19)**: Frontend dashboard (EZ Agent Hub).
- **Tailwind CSS 4**: Modern styling for the dashboard.
- **Ollama**: Local LLM runner (defaults to `qwen2.5-coder:7b`).
- **Supabase**: Backend-as-a-Service for data persistence and authentication.
- **n8n**: Workflow automation and orchestration.

## Testing & QA
- **Maestro**: UI testing for Flutter/Mobile applications.
- **Playwright**: End-to-end testing for Web/Next.js applications.

## Communication & DevOps
- **Discord**: Real-time notifications and reporting via webhooks.
- **GitHub**: Source control and repo management for target products.
- **EZ-Connect**: Internal platform for bug tracking and reporting.
